# 成绩录入模块 — 测试用例报告

**测试日期:** 2026-05-08  
**测试环境:** Spring Boot 2.7.18 + Vue 3 + MySQL 8.0  
**测试数据:** 80名学生, 4个班级, 6场考试, 1440条成绩

---

## 一、功能测试

### TC01 - 下拉数据加载

| 项目 | 内容 |
|------|------|
| **步骤** | 进入成绩录入页面 |
| **预期** | 考试下拉显示6个考试，课程下拉显示3门课程，班级下拉显示4个班级 |
| **结果** | ✅ PASS — 考试6个、课程3门(语文/数学/英语)、班级4个(七年级1/2班、八年级1/2班) |

### TC02 - 加载成绩表

| 项目 | 内容 |
|------|------|
| **步骤** | 选择考试"第一次月考"、班级"七年级1班"、课程"语文"，点击"加载成绩表" |
| **预期** | 表格显示20名学生，已录入的成绩回填，未录入的显示空白 |
| **结果** | ✅ PASS — 班级20名学生加载成功，已有成绩正确回填 |

### TC03 - 手动录入新成绩

| 项目 | 内容 |
|------|------|
| **步骤** | 对未录入的学生输入成绩，点击"保存成绩" |
| **预期** | 保存成功，状态变为"已录入"，修改标记清除 |
| **结果** | ✅ PASS — code 200，成绩持久化到数据库 |

### TC04 - 修改已有成绩(重新保存)

| 项目 | 内容 |
|------|------|
| **步骤** | 加载已有成绩表 → 修改分数 → 再次保存 |
| **预期** | 保存成功，不报"存在重复的成绩记录"错误 |
| **结果** | ✅ PASS — 带id的记录走更新逻辑，无id的重复记录自动跳过 |

### TC05 - 标记缺考

| 项目 | 内容 |
|------|------|
| **步骤** | 勾选某学生的"缺考"复选框，保存 |
| **预期** | 成绩清空，absent=1，保存后该生不计入平均分 |
| **结果** | ✅ PASS — 缺考学生 absent=1, score=null，统计中正确排除 |

### TC06 - 成绩输入范围限制

| 项目 | 内容 |
|------|------|
| **步骤** | 查看分数输入框的max属性 |
| **预期** | 使用当前课程fullScore(语文100分)，而非硬编码100 |
| **结果** | ✅ PASS — max值动态取自 courseInfo.fullScore |

### TC07 - 实时统计

| 项目 | 内容 |
|------|------|
| **步骤** | 录入成绩后查看底部统计栏 |
| **预期** | 实考人数、缺考人数、平均分、及格率正确计算 |
| **结果** | ✅ PASS — 及格率使用 courseInfo.passScore(60分) 动态计算 |

### TC08 - 下载导入模板

| 项目 | 内容 |
|------|------|
| **步骤** | 点击"批量导入"→"下载模板" |
| **预期** | 下载 .xlsx 文件，包含该班学生学号、姓名及空白的成绩/备注列 |
| **结果** | ✅ PASS — 4152字节有效xlsx文件，含20名学生数据 |

### TC09 - Excel批量导入

| 项目 | 内容 |
|------|------|
| **步骤** | 准备含学号、姓名、成绩的xlsx文件，点击"确定导入" |
| **预期** | 按学号匹配学生，正确解析成绩，返回成功/失败统计 |
| **结果** | ✅ PASS — 3条全部导入成功(2条有成绩、1条缺考) |

### TC10 - Excel导入错误处理

| 项目 | 内容 |
|------|------|
| **步骤** | Excel中包含不存在的学号、非法成绩格式 |
| **预期** | 不存在的学号报"不存在"，非法格式报"格式错误"，成功条数不受影响 |
| **结果** | ✅ PASS — 逐行解析，单行失败不影响其他行 |

### TC11 - 成绩统计联动

| 项目 | 内容 |
|------|------|
| **步骤** | 录入成绩后，前往"成绩统计"页面，选择同一考试和科目查看统计 |
| **预期** | 统计数据(平均分、及格率、分数段)与录入数据一致 |
| **结果** | ✅ PASS — totalCount=80, avgScore=81.82, passRate=100% |

### TC12 - 大屏展示联动

| 项目 | 内容 |
|------|------|
| **步骤** | 录入成绩后，前往"数据大屏"，选择相同考试+科目 |
| **预期** | 概览卡片、分数分布饼图、班级对比、Top10排名、三率分析、趋势图均正常渲染 |
| **结果** | ✅ PASS — 5个图表全部渲染，阈值(满分/及格线等)动态传递到前端 |

## 三、Excel导入格式规范

| 列名 | 必填 | 说明 |
|------|------|------|
| 学号 | ✅ | 与系统中学号一致 |
| 姓名 | - | 仅供参考，实际按学号匹配 |
| 成绩 | - | 数字(支持小数)，为空或含"缺考"则标记缺考 |
| 备注 | - | 可选备注信息 |
